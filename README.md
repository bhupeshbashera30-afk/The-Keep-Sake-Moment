# Keepsake Moments

Keepsake Moments is a premium event and gifting website built with React, Vite, TypeScript, Tailwind CSS, and Supabase.

## Pages
- Home
- About
- Contact
- Photobooth Rental
- Hampers & Flower
- Dinner Night
- Event & Decor
- Packages

## Tech stack
- React
- Vite
- TypeScript
- Tailwind CSS
- React Router DOM
- Supabase
- Lucide React

## Local setup
1. Clone the repository
2. Copy `.env.example` to `.env`
3. Add your Supabase credentials
4. Run `npm install`
5. Run `npm run dev`

## Build
Run `npm run build`

## Deploy
Push to GitHub and deploy on Netlify or Vercel.

## Supabase setup
Run the SQL in `supabase/schema.sql` inside the Supabase SQL editor.
